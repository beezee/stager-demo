Stager takes the pain out of coordinating staging environments when working on web applications in a team setting.

Automated Staging environments, integrated with your engineering process

Just focus on shipping, let staging happen by default

Github Integration

If you are using Github, Stager will watch your Pull Requests and make sure each has its own staging environment, even placing a link on the related PR for quick access.

Modular and Customizable

Not using Github? Integrations are trivial to implement and well-documented , contributions are gladly welcomed.

Unobtrusive and Unopinionated

Being Docker based, Stager does not care what your application is or how it is set up. If you can run it in a Docker container Stager can run it.
